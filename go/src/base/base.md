1. golang命令
  -
    build: 编译包和依赖
    clean: 移除对象文件
    doc: 显示包或者符号的文档
    env: 打印go的环境信息
    bug: 启动错误报告
    fix: 运行go tool fix
    fmt: 运行gofmt进行格式化
    generate: 从processing source生成go文件
    get: 下载并安装包和依赖
    install: 编译并安装包和依赖
    list: 列出包
    run: 编译并运行go程序
    test: 运行测试
    tool: 运行go提供的工具
    version: 显示go的版本
    vet: 运行go tool vet
